# BASIC LINUX COMMANDS:
- CD (CHANGE DIRECTORY FORWARD)
- ../ (CHANGE DIRECTORY BACKWARD)
- PWD (PRESENT WORKING DIRECTORY)
- LS (LIST OF FILES)
- RM (REMOVE)
- MKDIR (REMOVE DIRECTORY)
- RMDIR (REMOVE DIRECTORY)
- TOUCH (CREATE A NEW FILE)
- NANO (EDIT A FILE)
- CAT (FILE DISCRIPTION)
